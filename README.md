# top-pokemon-master

## プロジェクト概要
Elm Pages フレームワークを使った静的サイトジェネレーターです。Elm 0.19.1 でフロントエンドを構築し、Netlify にデプロイできます。ページは `app/Route/` 以下に追加し、スタイルは `public/style.css` で管理します。

## セットアップ
- 依存関係は `npm install` でインストールします。初回実行時に elm-tooling も自動で導入されます。

## 開発サーバーの起動と確認
1. `npm start` を実行してローカル開発サーバーを起動します（ポート 1234）。
2. ブラウザでアクセスする前に `ping localhost` で応答を確認し、サーバーが起動していることを必ず確かめてください。
3. アプリの確認は `http://localhost:1234/` を開きます。

## ビルドとテスト
- 本番ビルド: `npm run build` を実行すると `dist/` に成果物が生成されます。
- テスト実行: `npm test` で Elm のユニットテストおよび統合テストを実行します。変更を完了とする前にテストと静的解析がすべて通っていることを確認してください。

## 開発時の補足
- Elm の静的解析結果を確認し、警告やエラーを解消してから動作確認を行ってください。
- データ取得が必要な場合は `custom-backend-task.ts` にカスタム BackendTask を定義できます。
