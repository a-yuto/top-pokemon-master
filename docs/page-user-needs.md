# 各ページが解決するユーザー課題

## トップページ（`/`）
- **抱えている課題**: どのツールを使えば自分の悩みが解決するのか分からず、目的のコンテンツに辿り着けない。
- **解決アプローチ**: キャッチコピーとともに各ページの用途を一文で説明したリンクを一覧化し、初回訪問でも必要な機能へ迷わずアクセスできるようにする。

## 素早さクイズ（`/speed-quiz`）
- **抱えている課題**: 対戦中に「どちらが先手を取れるか」を瞬時に判断できず、読み負けてしまう。
- **解決アプローチ**: 種族値モードと実数値モードを切り替えながら反復練習できるクイズ形式を提供し、正答率表示と解説で試合前に判断力を鍛えられるようにする。

## 素早さ表（`/speed-table`）
- **抱えている課題**: 調整案を考える際に主要ポケモンの素早さ実数値を毎回計算・比較する手間が大きい。
- **解決アプローチ**: 使用率上位50匹の実数値を努力値・性格・持ち物条件別に可視化し、並べて比較できる散布図で抜き調整の検討を効率化する。

## ポケモン図鑑（`/pokedex`）
- **抱えている課題**: 構築候補の長所・短所を素早く把握できず、対戦用パーティの検討が進まない。
- **解決アプローチ**: 種族値・タイプ・合計値をカードに集約した一覧を提供し、興味があるポケモンを比較しながら候補を絞り込めるようにする。

## ポケモン図鑑 詳細（`/pokedex/:id`）
- **抱えている課題**: 個別ポケモンの詳細データを参照できる場所がなく、知りたい情報を深掘りできない。
- **解決アプローチ**: 事前生成された詳細ページの枠組みを用意し、公開済みIDでは基本情報を提示、未対応IDには準備状況を示して開発の進行度を透明化する。

## ポケモン検索（`/search`）
- **抱えている課題**: 条件からポケモンを探したいが、現状のツールでは必要なフィルタリングができない。
- **解決アプローチ**: プレースホルダーとして検索ページの土台を公開し、将来的な機能追加予定を明示してユーザーの期待値を調整する。
